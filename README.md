# SLTableView

###使用方式

在项目根目录build.gradle中加入

```Java
allprojects {
    repositories {
        jcenter()
        maven(){
            url 'https://dl.bintray.com/xiaoshanlin000/maven'
        }
    }
}
```
在Module build.gradle中加入
```Java
compile 'com.shanlin.library.sltableview:library:1.0.1'
```
